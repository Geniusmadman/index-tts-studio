# Changelog

所有重要变更都会记录在这里。版本号建议遵循 `主版本.次版本.修订号`。

## [0.1.0] - 2026-07-09

### Added

- 重构 Gradio WebUI 为「智能配音工作台」。
- 新增三种同级音色来源：本地上传、网络素材、常用音色库。
- 新增本地音视频转参考音色能力，统一转换为 24kHz 单声道 WAV。
- 新增常用音色保存与下拉复用。
- 新增网络素材解析能力，依赖 `yt-dlp`。
- 新增网络素材起止时间截取；默认前 60 秒，最大跨度 60 秒。
- 新增大模型/本地规则文本情绪分析。
- 新增长文本分段情绪分析、逐段生成、自动合并音频。
- 新增大模型配置持久化和代理开关。
- 新增反向代理子路径启动参数 `--root_path`。
- 新增音乐风深色主题、动效和可读性修复。
- 新增派生项目说明文档。
- 新增本地音视频上传大小提示和 1024 MB 默认限制。
- 新增本地上传/网络素材处理中的状态反馈。
- 新增本地上传/处理取消入口。

### Fixed

- 修复普通浏览器中 Gradio 默认主题覆盖导致的浅底白字问题。
- 修复 launchd 启动时找不到 `ffmpeg` 导致上传音视频无法处理的问题。
- 补充私有反向代理上传大文件被 Nginx 默认大小限制拦截时的部署说明。
- 修复 Gradio 右上角提示浅底白字导致不可读的问题。

### Changed

- 默认 UI 语言固定为中文。
- `.gitignore` 增加模型权重、日志、生成音频、用户音色、密钥和本机部署文件忽略规则。

## 版本规划
